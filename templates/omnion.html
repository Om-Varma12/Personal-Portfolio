Detailed Description

OmNion is a smart, LLM-powered personal chatbot assistant designed to answer user queries about me and my projects. Built using Flask, Together.ai APIs, HTML, CSS, and JavaScript, it blends a clean frontend with a powerful backend for natural, conversational interactions.

It leverages large language models to understand open-ended questions, provide detailed responses, and simulate a human-like assistant with memory and context capabilities. Whether you're curious about my work, tech stack, or latest innovations ‚Äî OmNion has the answers.

OmNion is an evolved extension of Quirky ‚Äî the original intent-based chatbot built by me ‚Äî but now supercharged with Together.ai's LLMs, enabling more intelligent and flexible conversations beyond predefined tags.
