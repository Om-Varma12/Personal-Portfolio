Problem Statement

Students often go through long PDFs ‚Äî textbooks, notes, or assignments ‚Äî just to find answers to a few specific questions. This can be time-consuming and frustrating, especially during exam time or while revising. There is a need for a smart, easy-to-use tool that helps students quickly get answers from their own study material without reading everything.

Detailed Description

Chaplify is a smart PDF-based chatbot assistant designed to help students get instant answers from their own study material. Instead of scrolling through long textbooks or notes, students can simply upload a PDF and ask questions ‚Äî Chaplify reads the document and responds using natural language.

Powered by Together.ai's large language models and built with Flask, Chaplify combines LLM intelligence with a clean frontend interface. It uses PDF parsing, intelligent chunking, vector embeddings, and semantic search to find the most relevant context before generating an accurate and helpful response.

Chaplify makes self-study faster and smarter by turning any document into an interactive knowledge assistant. Whether it's for revision, assignments, or quick concept clarifications ‚Äî Chaplify is there to help.

Reflections & Improvements

Building Chaplify was my first deep dive into combining document processing with real-time, LLM-powered conversations. I learned how to extract and structure data from PDFs, manage large context windows, and feed relevant information to the model in a meaningful way. Working with Together.ai's API helped me understand prompt design, latency handling, and error fallback techniques. This project also strengthened my backend skills with Flask and improved how I build clean, functional frontends using fetch and JavaScript. Going forward, I'd like to enhance Chaplify with PDF summarization, multi-file support, and user session memory for even smoother conversations.
