Detailed Description

Quirky is a simple, intent-based chatbot built using Flask, HTML, CSS, and JavaScript. It uses a custom JSON file to classify user messages into predefined tags like greetings, info, or fallback, and responds with dynamic, conversational replies.

Designed as a learning project, Quirky focuses on understanding backend logic, data flow, and real-time interaction without relying on heavy NLP models. It offers a smooth, interactive chat interface and serves as a foundation for future AI integrations.
