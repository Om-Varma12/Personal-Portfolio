Problem Statement

Many individuals struggle to keep track of their daily spending, often leading to overspending and poor financial awareness. Traditional methods like spreadsheets are manual and time-consuming. PennyWise.io solves this by providing a simple, digital way to log and categorize expenses, giving users instant insights into their financial habits.

Screenshots

Detailed Description

PennyWise.io is a personal finance web application built to help users track their income and expenses in a clean, intuitive interface. Designed as a full-stack web app using Flask, HTML, CSS, and JavaScript, it enables users to add, view, and delete transactions in real-time. Each transaction is categorized (Food, Transport, Bills, etc.) and dynamically reflected in a balance summary.

The goal is to encourage financial discipline by offering users a visual breakdown of their spending behavior. The project combines a responsive front end with a Python-based backend, and can be deployed online for personal use.

Reflections & Improvements

Creating PennyWise.io was a valuable exercise in full-stack web development and UI design. It helped reinforce concepts like data handling with Flask, form validation, and real-time DOM updates with JavaScript. Future enhancements could include secure user accounts, mobile responsiveness, and visual analytics using libraries like Chart.js.
